# Learning From Data Analysis Bootcamp  

This repository documents my learning journey from the **Data Analysis Bootcamp**, where I explored tools and techniques essential for modern data-driven decision-making.  

## 📘 Overview  
Throughout the bootcamp, I gained hands-on experience with:  
- **Excel** – Data cleaning, pivot tables, dashboards  
- **Pandas (Python)** – Data manipulation, analysis, and visualization  
- **Power BI** – Interactive dashboards and business intelligence reports  
- **Python** – Scripting, automation, and data visualization  
- **SQL** – Querying, joins, aggregations, and database management  
- **Web Scraping** – Extracting and structuring data from websites  

## 🎯 Objectives  
- Build a strong foundation in data analysis and visualization  
- Apply analytical skills to real-world datasets  
- Create portfolio-ready projects showcasing technical and business insights  

## 📂 Repository Structure  
- `Excel/` → Practice sheets and dashboards  
- `Pandas/` → Python notebooks for data manipulation  
- `PowerBI/` → BI reports and visualizations  
- `Python/` → Scripts and analysis projects  
- `SQL/` → Query files and database exercises  
- `Web_Scraping/` → Scripts for extracting online data  

## 🚀 Skills Gained  
- Data cleaning and preprocessing  
- Exploratory data analysis (EDA)  
- Visualization and storytelling with data  
- Business intelligence reporting  
- Automation and workflow optimization
